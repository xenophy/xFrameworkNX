  createDelegate(Object obj, [Array args], [Boolean/Number appendArgs]) : Function

配列に指定された関数をコールバック関数として登録します。

また、第３引数の指定を行うことで、コールバック関数の引数を変更することができます。

var sayHi = function(name){
    console.log('Hi, ' + name + '.');
}

var f = sayHi.createDelegate({foo: 'bar'}, ['Fred']);

f();
パラメータ:
  • obj:Object

    • スコープオブジェクト

  • args:Array
    • (オプション)

      引数の上書き(デフォルトではコールバック関数から渡される引数)

  • appendArgs:Boolean/Number
    • (オプション)

      true: 第２引数で指定した引数を、コールバック関数にデフォルトで渡される引数の後ろに追加。

      false: 第２引数を配列として渡す。

      数値: コールバック関数の引数内に、指定した数値番目に引数を挿入（数値 == N とすると第N+1引数)

戻り値:
  • Function

    • 新たに生成された関数

  createInterceptor(Function fcn, [Object scope]) : Function

引数に設定する関数の戻り値が、trueの場合のみ設定した関数を実行することができます。

条件によって関数を呼び出したくない場合などに使用します。

var sayHi = function(name){
    console.log('Hi, ' + name);
}

sayHi('Fred');

var sayHiToFriend = sayHi.createInterceptor(function(name){
    return name == 'Brian';
});

sayHiToFriend('Fred');

sayHiToFriend('Brian');
パラメータ:
  • fcn:Function

    • 元の関数が実行される前に実行される関数(判定などの関数で、戻り値が真偽型)

  • scope:Object
    • (オプション)

      呼び出されるインターセプタ関数のスコープオブジェクトを指定します。(初期値：元の関数またはglobalオブジェクト)

戻り値:
  • Function

    • 新たに生成された関数

  createSequence(Function fcn, [Object scope]) : Function

元の関数を呼び出した後に、任意の処理を追加して呼び出します。

var sayHi = function(name){
    console.log('Hi, ' + name);
}

sayHi('Fred');

var sayGoodbye = sayHi.createSequence(function(name){
    alert('Bye, ' + name);
});

sayGoodbye('Fred');
パラメータ:
  • fcn:Function

    • 元の関数が実行された後に実行される関数

  • scope:Object
    • (オプション)

      呼び出されるシーケンシャル関数のスコープオブジェクトを指定します。(初期値：元の関数またはglobalオブジェクト)

戻り値:
  • Function

    • 新たに生成された関数

  defer(Number millis, Object obj, [Array args], [Boolean/Number appendArgs]) : Number

ミリ秒で設定した時間後、この設定された関数を呼び出します。

パラメータ:
  • millis:Number

    • 待機させる時間をミリ秒で設定(内部でsetTimeout使用)、もし0を設定した場合は、即時実行

  • obj:Object

    • 関数のスコープオブジェクトを指定します。

  • args:Array
    • (オプション)

      引数の上書き(デフォルトではコールバック関数から渡される引数)

  • appendArgs:Boolean/Number
    • (オプション)

      true: 第２引数で指定した引数を、コールバック関数にデフォルトで渡される引数の後ろに追加。

      false: 第２引数を配列として渡す。

      数値: コールバック関数の引数内に、指定した数値番目に引数を挿入（数値 == N とすると第N+1引数)

戻り値:
  • Number

    • clearTimeoutで利用できるid
