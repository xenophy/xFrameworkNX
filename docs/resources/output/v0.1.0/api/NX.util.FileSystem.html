                                    pathinfo(String path, [String options]) : Mixed value

                                  pathinfo()は、pathに関する情報を有するオブジェクトを返します。

                                  パラメータ:
                                  • path:String

                                    • 調べたいパス

                                  • options:String
                                    • (オプション)

                                      どの要素を返すのかをオプションのパラメータoptionsで指定します。

                                      これは'PATHINFODIRNAME'、'PATHINFOBASENAME'、'PATHINFOEXTENSION'および'PATHINFOFILENAME'の組み合わせとなります。 デフォルトではすべての要素を返します。

                                  戻り値:
                                  • Mixed value

                                    • 以下の要素を含むオブジェクトを返します。dirname(ディレクトリ名)、basename(ファイル名)そして、もし存在すればextension(拡張子)。

                                      optionsを使用すると、すべての要素を選択しない限りこの関数の返り値は文字列となります。
