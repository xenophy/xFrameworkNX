NX.util.JSON クラス

Douglas Crockford's によって作られたjson.js(http://www.json.org/js.html)を変更したクラスです。


このクラスはシングルトンパターンで実装されています。直接生成することはできません。

メソッド

メソッド 定義クラス
  decode(String json) : Object

JSON文字列を解析してオブジェクトにします。 JSON文字列が無効であれば、この関数はSyntaxErrorを投げます。

パラメータ:
  • json:String

    • JSON文字列

戻り値:
  • Object

    • デコードされたオブジェクト

オブジェクトや配列、またはその他の値をJSON文字列に変換します。

パラメータ:
  • o:Object

    • エンコードするオブジェクト

戻り値:
  • String

    • オブジェクトや配列、またはその他の値をJSON文字列に変換します。
