NX.util.MixedCollection クラス

数値のインデックスとキーとそれらを操作する時のイベント発生させ管理するクラスです。

メソッド

メソッド 定義クラス
  MixedCollection([Boolean allowFunctions], [Function keyFn]) : void

コンストラクタ

パラメータ:
  • allowFunctions:Boolean
    • (オプション)

      trueの場合、addAllメソッドで関数オブジェクトを追加可能にします。(初期値 false)

  • keyFn:Function
    • (オプション)

      getKeyとして呼び出される関数を設定します。 アイテム内のパラメータを戻り値として返却しキーとすることができる関数です。 この関数は、アイテム内のキーに取り出そうとしたキーが存在しない場合に適用されます。

戻り値:
  • void
NX.util.MixedCollection
  add(String key, Mixed value o) : Object

アイテムを追加します。

完了した時点でaddイベントを発火します。

パラメータ:
  • key:String

    • キー オブジェクトのidキーを、内部のidとして利用します。 getKeyを実装するか、コンストラクタのkeyFnを指定した場合は、そちらを利用します。

  • o:Mixed value

    • 追加するアイテム

戻り値:
  • Object
